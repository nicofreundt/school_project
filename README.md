# Schul-Projekt - Feedback_Django #

## VERSIONS ##
- Python 3.10.2
- Django 4.0.2
- MariaDB 10.7.3

## Creating new pipenv (venv) using Linux (Ubuntu) ##
- ```pipenv --python 3.10 install django```
- ```pipenv shell```
- ```curl -sS <https://bootstrap.pypa.io/get-pip.py> | python3.10```
- ```pip install django```

## Creating new pipenv (venv) using Windows ##
- ```pipenv --python 3.10 install django```
- ```pipenv shell```
